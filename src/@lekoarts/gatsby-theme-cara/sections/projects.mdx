## 🤖 Automation 🏭

<ProjectCard
    title="Digicel Router Scraper"
    link="https://github.com/jefroy/digicel-router-scraper"
    bg="linear-gradient(to right, #f39c12, #34495e)"
    >
Using Selenium and Python 3, this script will run every 15 minutes and login to your digicel router,
scan the ports table, and output the results to a json file.
This was used in conjunction with a discord bot to allow a discord server to get access to the public ip and port for port 25565 allowing users to play on their homebrewed Minecraft server.
</ProjectCard>
